
# Pug Generator

This python script helps with generating pug files based on user description.


## Run Project

Clone the project

```bash
  git clone https://github.com/y0dev/pug_generator
```

Go to the project directory

```bash
  cd pug_generator
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Run the script

```bash
  python main.py
```
